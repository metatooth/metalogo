# Meta**logo**

Automates the calling of a BASH script to produce the Meta**tooth** logo.

## Getting Started

### Prerequisites

#### Ubuntu 18.04 LTS

Use it!

### Install

```
$ git clone https://bitbucket.org/metatooth/metalogo
$ cd metalogo
$ cmake -H. -B_builds
$ cmake --build _builds
```
